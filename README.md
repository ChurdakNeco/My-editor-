# My editor — copy of VIM for Rust

It's a super fust editor based on ANSI escape sequences
for learning

## Keybindings
 ________________________________
|       Key      |     Action     |
|----------------|----------------|
| Ctrl+Q         | Quit           |
| Ctrl+S         | Save           |
| Ctrl+Z         | Undo           |
| Ctrl+Y         | Redo           |
| Ctrl+F         | Search         |
| Ctrl+C         | Copy           |
| Ctrl+X         | Cut            |
| Ctrl+V         | Paste          |
| Ctrl+A         | Select al      |
| Ctrl+D         | Delete line    |
| Ctrl+K         | Clear buffer   |
| Shift+Arrows   | Select text    |
| Home/End       | Line start/end |
| PageUp/Down    | Page up/down   |
